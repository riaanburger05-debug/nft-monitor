# GroeiGids op die nuwe laptop

## Eenmalige opstelling

1. Meld by dieselfde OneDrive-rekening aan.
2. Wag dat `Desktop\Agent\nft` volledig sinkroniseer.
3. Regskliek die `nft`-gids en kies **Always keep on this device**.
4. Installeer:
   - Codex
   - Google Chrome
   - Git
   - Python 3, met **Add Python to PATH** gemerk
5. Open `C:\Users\<jou-naam>\OneDrive\Desktop\Agent` in Codex.

## Begin GroeiGids

Dubbelklik:

```text
nuwe-laptop-begin.bat
```

Die venster wys:

- Die adres vir die nuwe laptop.
- Die korrekte Wi-Fi-adres vir jou foon.

Hou die venster oop terwyl jy die app gebruik.

## Belangrike leers

```text
tenk-monitor-ux.html
Images\nft-coach-reference.png
groeigids_ux_ui_improvement_brief_for_codex.md
```

## Kontrole voordat die ou laptop uitgevee word

- Die UX-prototipe maak op die nuwe laptop oop.
- Die Coach-beeld vertoon.
- Die app maak op jou foon oop.
- Bestaande tenkdata verskyn vanaf Firebase.
- Die volledige `nft`-gids is beskikbaar sonder internet.
